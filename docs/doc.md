1. type1(cash)

    * 记录号, 证券代码, 证券名称, 可用数量, 证券数量, 成本价, 最新价, 盈亏, 市值, 资产账户, 交易板块, 交易账户, 盈亏比例% 
3. type3(cash)
    * 币种, 余额, 可用, 可取, 参考市值, 资产, 盈亏
    * 证券代码, 证券名称, 证券数量, 可卖数量, 冻结数量, 在途数量, 成本价, 当前价, 最新市值, 浮动盈亏, 盈亏比例, 交易所名称, 股东代码, 资金账号
